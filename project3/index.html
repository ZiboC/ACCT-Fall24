Project 3: Synthetic Relationships

Project Overview

The system consists of two main components: the Light Tower and the Light-Seeking Robot, both of which communicate through cloud-based data feeds for synchronized interaction. The Light Tower is equipped with a servo motor, an LED light, and a laser range sensor. The LED light serves as a detectable signal for the robot, while the servo motor rotates upon receiving a signal from the robot. The laser range sensor monitors the robot's proximity and triggers specific actions when the robot is near. The Light-Seeking Robot features a two-wheel drive, a distance sensor, and a light sensor. It autonomously navigates the environment, using the distance sensor to avoid obstacles. When the robot's light sensor detects the LED light from the tower, it sends a feed to the cloud, causing the servo on the Light Tower to rotate. As the robot approaches the tower, the laser range sensor sends another feed to the cloud, prompting the robot to twist twice and then move backward.
